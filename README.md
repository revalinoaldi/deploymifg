# Nabila API

Nabila API revamp using FastAPI

### Basic Requirements:
- fastapi 0.63.0
- uvicorn
- Redis
- mysqlclient 2.0.2
- DBUtils
- passlib
- python-jose


### Run API

- Run API command: 
```bash
    uvicorn nabila:app
```

- Run API command with hot reload:
```bash
    uvicorn nabila:app --reload
```
